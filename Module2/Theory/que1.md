1. Difference between Web Designer and Web Developer 

Web Designer:

A Web Designer focuses on the visual appearance, layout, and user experience of a website.
They make sure the site looks attractive and is easy to use.

What They Do :

Create layouts, wireframes, and mockups of websites.

Decide color schemes, fonts, and typography.

Design buttons, menus, banners, icons.

Ensure the website is responsive (works on mobile, tablet, desktop).

Focus on UI (User Interface) and UX (User Experience).

Tools They Use :

Graphic/UI tools: Figma, Adobe XD, Photoshop, Sketch, Canva.

Basic coding knowledge: HTML, CSS (sometimes).

think of a Web Designer as the architect + interior designer of a website. They plan and design how it should look and feel.

Web Devloper :

A Web Developer builds the functionality and logic of the website.
They make sure the site works properly and can handle user interactions.

Types of Web Developers :

Frontend Developer → Builds the part of the site you see (HTML, CSS, JavaScript, React, etc.).

Backend Developer → Builds the server, database, and logic (Node.js, PHP, Python, Java, etc.).

Full-Stack Developer → Handles both frontend and backend.

 What They Do :

Convert the designer’s mockups into real, working websites.

Add functionality (forms, login, payment systems, dynamic data).

Connect website to a database & server.

Ensure the site is fast, secure, and bug-free.

Tools They Use :

Languages: HTML, CSS, JavaScript, Python, PHP, Java, C#, etc.

Frameworks: React, Angular, Node.js, Django, Laravel.

Databases: MySQL, MongoDB, PostgreSQL.

 Think of a Web Developer as the engineer + builder of a website. They construct and run the site so it actually works.
